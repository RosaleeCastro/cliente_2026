calcularTotal(subtotal, descuento, iva) → {number}

Calcula el total final de una compra.

El total final se obtiene restando el descuento al subtotal y sumando después el IVA correspondiente.

Fórmula utilizada:

total = subtotal - descuento + iva

gestionarFormulario(evento) → {void}

Gestiona el envío del formulario principal.

Esta función realiza el flujo completo de la aplicación:

  1. Cancela el envío tradicional del formulario.
  2. Lee los valores introducidos por el usuario.
  3. Valida los datos.
  4. Calcula el subtotal.

validarDatos(precio, cantidad, iva, descuento) → {boolean}

Comprueba si los datos introducidos en el formulario son válidos.

Un formulario se considera válido cuando:

  • el precio es mayor o igual que 0;
  • la cantidad es mayor que 0;
  • el IVA es mayor o igual que 0;
  • el descuento es mayor o igual que 0;
  • todos los valores son numéricos.
Parameters:
Name Type Description
precio number

Precio unitario introducido por el usuario.
cantidad number

Cantidad de unidades introducida.
iva number

Porcentaje de IVA introducido.
descuento number

Porcentaje de descuento introducido.
Source:
  • app.js, line 67
Returns:

Type
boolean
Examples

// Devuelve true
validarDatos(100, 0, 21, 10);
// Devuelve false
